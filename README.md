## Descrição
Este é um programa em Python que baixa vídeos do YouTube. Ele utiliza a biblioteca PySimpleGUI para a interface gráfica e pytube para o download dos vídeos.

## Requisitos
- Python 3.x
- PySimpleGUI
- pytube

### 1. Instale as Bibliotecas:
